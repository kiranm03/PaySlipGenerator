This project will generate the payslip with tax details by taking employee details as input. This is an console application, it will accept CSV file input with employee details and generate payslips to another CSV file.


DEVELOPMENT PLATFORM
----------------------
- .NET Core 3.0
- Visual Studio 2019

INPUT & OUTPUT
----------------
- A sample input file is available in the solution folder. File name:Employee_Data.csv
- Output file is an CSV and will be generated bin folder of this application.
